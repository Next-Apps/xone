Tutorial

Building an app with Xone

This tutorial is a step by step explanation of most features provided by Xone and will be updated continuously.


Xone Installation Requirements: Node.js

Install Xone via NPM (Node Package Manager): 

                    > npm install -g xone


Create blank project folder an run from it: 

                    > xone create


Build the new created project: 

                    > xone build


Start local webserver: 

                    > xone server


Open your browser and goto:
  • http://localhost/app to run development environment
  • http://localhost/bin/www to run production build ("www" is the default platform)
